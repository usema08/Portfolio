Latest Project

Evently

Node.js | Next.js | TypeScript | TailwindCSS | Stripe | Zod | React Hook Form | Shadcn | uploadthing
Built on Next.js 14, the events application stands as a comprehensive, full-stack platform for managing events. It serves as a hub, spotlighting diverse events taking place globally. Featuring seamless payment processing through Stripe, you have the capability to purchase tickets for any event or even initiate and manage your own events.

E-commerce Website

Posts

Node.js | React.js | Express.js | JavaScript | BootStrap | Docker | Kubernetes | Ingress Nginx | Skaffold
The app uses event-driven architecture with an event bus for seamless posting and commenting. Powered by microservices in Node.js, it ensures scalability and responsiveness. Leveraging Docker and Kubernetes, microservices are efficiently deployed and managed. Users create posts triggering real-time updates and notifications. Experience dynamic content sharing and interaction on our modern platform.

Dashboard

React.js | JavaScript | CSS
The "Social Media Dashboard" mini project is a customizable web application that tracks social media performance metrics and offers color theme options. Users can toggle between different themes and view key metrics like follower growth and post engagement rates. This dashboard helps users refine content strategies by identifying trends in growth and decline over time. It's responsive across devices for convenient monitoring and optimization of social media presence.
